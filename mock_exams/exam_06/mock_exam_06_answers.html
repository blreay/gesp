第1题 ×：(3, 5, 7) 是逗号表达式，取最后一个值 7，不是 15（三者之和）。
第2题 √：x = 0 是赋值表达式，将 x 赋值为 0，表达式的值为 0。0 为假，所以条件永远为假，不会输出 "yes"。
第3题 √：for(;;) 三个部分全部省略是合法的。条件部分省略时默认为真，等价于 for(;true;)，即无限循环。
第4题 √：65 是字符 'A' 的 ASCII 码。char c = 65; 等价于 char c = 'A';。cout << c 输出字符 A。
第5题 √：这就是 switch 的 fall-through 特性。如果没有 break，程序会继续执行下一个 case 的代码，直到遇到 break 或 switch 结束。
第6题 ×：C++ 中 ^ 是位异或运算符（XOR），不是求幂。2^3 的结果是 2 XOR 3 = 1（010 XOR 011 = 001），不是 8。C++ 中求幂应使用 pow(2, 3)。
第7题 ×：char 类型参与算术运算时会隐式提升为 int。c + 1 的结果类型是 int，不是 char。
第8题 ×：逗号表达式 (a++, a * 3)：先执行 a++（a 从 3 变为 4），再计算 a * 3 = 4 * 3 = 12。b=12，不是 10。
第9题 √：'A' 的 ASCII 码为 65，'a' 的 ASCII 码为 97。65 + 32 = 97，正好等于 'a' 的 ASCII 码值。
第10题 √：switch 的 case 标签必须是整型常量表达式（包括整数常量和字符常量），不能是变量、浮点数或其他非常量表达式。

编程题 1：小杨爬楼梯
思路：经典的斐波那契数列问题。设 f(n) 为爬到第 n 级台阶的方案数，则 f(1)=1, f(2)=2, f(n)=f(n-1)+f(n-2)。
因为最后一步可以从 n-1 级爬 1 级，或从 n-2 级爬 2 级。 
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    if (n == 1) {
        cout << 1 << endl;
        return 0;
    }
    if (n == 2) {
        cout << 2 << endl;
        return 0;
    }
    int a = 1, b = 2;
    for (int i = 3; i <= n; i++) {
        int c = a + b;
        a = b;
        b = c;
    }
    cout << b << endl;
    return 0;
}
编程题 2：区间内满足条件的数
思路：遍历 L 到 R 的每个数，判断是否能被 3 整除且不能被 5 整除，统计个数和总和。 
#include <iostream>
using namespace std;

int main() {
    int L, R;
    cin >> L >> R;
    int count = 0, sum = 0;
    for (int i = L; i <= R; i++) {
        if (i % 3 == 0 && i % 5 != 0) {
            count++;
            sum += i;
        }
    }
    cout << count << " " << sum << endl;
    return 0;
}
