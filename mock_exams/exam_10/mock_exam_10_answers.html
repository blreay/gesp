编程题 1：商店折扣
思路：先计算总价 total = p * n，然后按折扣规则分段判断实际应付金额。 
#include <cstdio>
using namespace std;

int main() {
    double p;
    int n;
    scanf("%lf %d", &p, &n);
    double total = p * n;
    double pay;
    if (total <= 100)
        pay = total;
    else if (total <= 300)
        pay = total * 0.9;
    else
        pay = total * 0.8;
    printf("%.2f\n", pay);
    return 0;
}
编程题 2：质数求和
思路：遍历 2 到 n 的每个数，判断是否为质数（只需检查 2 到 sqrt(i) 之间有无因数），是质数则累加。 
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int sum = 0;
    for (int i = 2; i <= n; i++) {
        bool isPrime = true;
        for (int j = 2; j * j <= i; j++) {
            if (i % j == 0) {
                isPrime = false;
                break;
            }
        }
        if (isPrime)
            sum += i;
    }
    cout << sum << endl;
    return 0;
}
