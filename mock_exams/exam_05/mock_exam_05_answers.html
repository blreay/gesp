第1题 ×：ENIAC 使用的是电子管（真空管）元件，不是晶体管。晶体管是第二代计算机的元件。
第2题 √：17/5=3（整数除法），3*5=15，17%5=2，15+2=17。这是整数除法的恒等式：(a/b)*b + a%b == a。
第3题 √：do-while 循环先执行循环体再判断条件，所以至少执行一次，即使条件一开始就为假。
第4题 √：在 printf 格式字符串中，%% 表示输出一个 % 字符。所以 printf("100%%") 输出 100%。
第5题 ×：for 循环没有大括号，只有 s += i; 是循环体。cout << s << " "; 虽然看起来有缩进，但它不在循环体内，只在循环结束后执行一次。此时 s=1+2+3+4+5=15，输出 15 （只输出一个数），而不是 1 3 6 10 15（每轮都输出）。考点：没有花括号时 for 只控制紧跟的一条语句。
第6题 ×：n=4050，n/10=405，405%10=5。结果是 5 而不是 0。n/10%10 提取的是十位数字，4050 的十位是 5。
第7题 ×：赋值运算符（如 +=）的优先级非常低，远低于算术运算符 +。在 C++ 运算符优先级表中，赋值运算符优先级几乎是最低的（仅高于逗号运算符）。
第8题 ×：输出是 1 2 3 4 5 （注意末尾有一个多余的空格），而不是 1 2 3 4 5（末尾无空格）。严格来说输出不完全相同，因为 5 后面还有一个空格。
第9题 ×：double 类型 2.9 赋值给 int 变量时会截断（不是四舍五入），所以 x 的值是 2，不是 3。
第10题 √："天河二号"由中国国防科技大学研制，曾在2013年6月至2015年11月连续6次获得 TOP500 排行榜冠军。
三、编程题参考答案
编程题 1：闰年统计
思路：遍历 Y1 到 Y2 的每个年份，判断是否满足闰年条件（能被4整除但不能被100整除，或能被400整除）。 
#include <iostream>
using namespace std;

int main() {
    int Y1, Y2;
    cin >> Y1 >> Y2;
    int count = 0;
    for (int y = Y1; y <= Y2; y++) {
        if ((y % 4 == 0 && y % 100 != 0) || y % 400 == 0)
            count++;
    }
    cout << count << endl;
    return 0;
}
编程题 2：数字排列
思路：提取三位数的百位 a、十位 b、个位 c，然后对 a、b、c 三个数字进行排序（从小到大），
组成新数 = 最小值×100 + 中间值×10 + 最大值。由于用整数输出，前导零自动省略。 
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;
    int a = N / 100;
    int b = N / 10 % 10;
    int c = N % 10;
    // 冒泡排序三个数
    if (a > b) { int t = a; a = b; b = t; }
    if (b > c) { int t = b; b = c; c = t; }
    if (a > b) { int t = a; a = b; b = t; }
    cout << a * 100 + b * 10 + c << endl;
    return 0;
}
