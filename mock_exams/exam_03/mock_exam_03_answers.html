二、判断题答案

第1题 ✓：这是 switch 的 fall-through 特性。没有 break 就会继续执行下一个 case 的代码，直到遇到 break 或 switch 结束。
第2题 ×：!0 的值是 1（逻辑非），所以 !0 + !0 = 1 + 1 = 2，不是 0。
第3题 ×：%d 用于输出整数，但 3.14 是 double 类型。将 double 传给 %d 是未定义行为，不能保证输出 3。正确做法应使用 %f 或先强制转换为 int。
第4题 ✓：这是经典的 1 到 100 求和：1+2+...+100 = 5050。
第5题 ✓：C++ 标准规定 case 标签后面必须是整型常量表达式（包括整数常量、字符常量、const 整型变量等），不能是普通变量。
第6题 ×：%%d 中，%% 输出一个字面百分号 %，后面的 d 是普通字符。所以输出是 %d（字面字符串），参数 10 被忽略（不被格式串消费）。不是输出 10。
第7题 ✓：while(1) 和 for(;;) 都是合法的无限循环写法。
第8题 ✓：花括号创建块作用域，内层定义的变量在离开花括号后生命周期结束，外层无法访问。
第9题 ✓：1 < 2 < 3 的计算过程是先算 1 < 2 = 1（true），再算 1 < 3 = 1（true），结果为 1。虽然结果碰巧为 true，但过程并不是数学上的"连续比较"。题目说法正确。
第10题 ×：if (a = 20) 是赋值而非比较！先将 a 赋值为 20，赋值表达式的值为 20（非零即真），所以条件为真，执行 cout << a 输出 20，不是 10。
三、编程题参考答案
编程题 1：星期计算
思路：利用取模运算。将星期几映射为 0~6 的偏移量再取模后加回来：(W - 1 + N) % 7 + 1。 
#include <iostream>
using namespace std;

int main() {
    int W, N;
    cin >> W >> N;
    int ans = (W - 1 + N) % 7 + 1;
    cout << ans << endl;
    return 0;
}
编程题 2：完全平方数统计
思路：枚举正整数 i，当 i*i 在 [L, R] 范围内时计数。 
#include <iostream>
using namespace std;

int main() {
    int L, R;
    cin >> L >> R;
    int count = 0;
    for (int i = 1; i * i <= R; i++) {
        if (i * i >= L)
            count++;
    }
    cout << count << endl;
    return 0;
}
