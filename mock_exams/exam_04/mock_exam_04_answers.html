C 项 a != b：辗转相除法不保证 a 和 b 相等时停止。
D 项 a % b != 0：当 a 能被 b 整除时停止，但此时 GCD 应为 b 而非 a，逻辑不对。

第1题 ✓：在 for 循环中，continue 跳过剩余循环体，然后执行自增表达式（i++），再判断条件。这是 for 循环中 continue 的特性。
第2题 ✓：在 while 循环中，continue 跳到循环条件判断处重新检查条件。注意这和 for 循环不同：for 循环的 continue 会先执行自增表达式。
第3题 ×：(int)3.9 是强制类型转换，将 double 截断为 int，结果是 3，不是 4。C++ 的强制转换是截断（向零取整），不是四舍五入。
第4题 ✓：char 类型本质上是整数类型（存储 ASCII 码），可以和整数进行加减运算。例如 'A' + 1 等于 66，即 'B'。
第5题 ✓：'A' 的 ASCII 码是 65，加 1 得 66，而 'B' 的 ASCII 码正好是 66。因此 'A'+1 == 'B' 为 true（值为 1）。
第6题 ×：x = 0，在 if 条件中 0 被视为 false，所以不会执行 cout << "hello"，不会输出任何内容。
第7题 ×：7/2 是整数除法，结果为 3（int 类型）。7.0/2 是浮点除法，结果为 3.5（double 类型）。两者的值不同。
第8题 ✓：这是 C++ 的"最近匹配"规则（也叫"悬挂 else"规则），else 与距离最近的、尚未匹配的 if 配对。可以用花括号 {} 来改变匹配关系。
第9题 ×：a/b 即 7/2 是整数除法，结果为 3（不是 3.5）。cout 输出 3，不是 3.5。
第10题 ×：'0' 是字符零，其 ASCII 码值是 48，不是 0。整数 0 对应的 ASCII 字符是 NUL（空字符），即 '\0'。
三、编程题参考答案
编程题 1：最大公约数
思路：使用辗转相除法（欧几里得算法）。不断用较大数除以较小数取余，直到余数为 0，此时除数即为最大公约数。 
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    while (b != 0) {
        int t = b;
        b = a % b;
        a = t;
    }
    cout << a << endl;
    return 0;
}
编程题 2：倒序输出数位
思路：不断取 N 的末位数字（N%10），拼接到结果 rev 中（rev = rev*10 + N%10），然后去掉 N 的末位（N/=10）。由于 rev 是整数，前导零自动消失。 
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;
    int rev = 0;
    while (N > 0) {
        rev = rev * 10 + N % 10;
        N /= 10;
    }
    cout << rev << endl;
    return 0;
}
