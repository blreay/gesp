三、编程题参考答案
编程题 1：小杨的零花钱
思路：每月净存 m - s 元，需要 ⌈p / (m-s)⌉ 个月，即向上取整。
可以用整数公式 (p + (m-s) - 1) / (m-s) 实现向上取整，也可以用循环累加。 
#include <iostream>
using namespace std;

int main() {
    int p, m, s;
    cin >> p >> m >> s;
    int save = m - s;   // 每月净存款
    int months = 0;
    int total = 0;
    while (total < p) {
        total += save;
        months++;
    }
    cout << months << endl;
    return 0;
}
编程题 2：数字之和
思路：不断取 n % 10 得到个位数字并累加，然后 n /= 10 去掉个位，直到 n 变为 0。 
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int sum = 0;
    while (n > 0) {
        sum += n % 10;
        n /= 10;
    }
    cout << sum << endl;
    return 0;
}
