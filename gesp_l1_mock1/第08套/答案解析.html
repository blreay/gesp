五、编程题参考程序
编程题 1：长方形周长与面积
#include <iostream>
using namespace std;
int main() {
    long long a, b;
    cin >> a >> b;
    cout << 2 * (a + b) << " " << a * b << endl;
    return 0;
}
要点：周长 2*(a+b)，面积 a*b。a、b 可达 10000，面积最大 1 亿，int 够用，用 long long 更稳妥。
编程题 2：数字三角形
#include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= i; j++) {
            if (j > 1) cout << " ";  // 数字间一个空格，行首不加
            cout << j;
        }
        cout << endl;               // 每行换行
    }
    return 0;
}
要点：双重循环，第 i 行打印 1~i；用 if(j>1) 控制空格位置，保证每行末尾无多余空格。
