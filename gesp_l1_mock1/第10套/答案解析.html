五、编程题参考程序
编程题 1：区间内 7 的倍数计数
#include <iostream>
using namespace std;
int main() {
    long long a, b;
    cin >> a >> b;
    long long cnt = 0, sum = 0;
    for (long long i = a; i <= b; i++)
        if (i % 7 == 0) { cnt++; sum += i; }
    cout << cnt << " " << sum << endl;
    return 0;
}
要点：遍历区间，用 i%7==0 判断 7 的倍数，同时计数与求和。区间最大 100 万，循环可在时限内完成；和较大，用 long long。
编程题 2：数字反转
#include <iostream>
using namespace std;
int main() {
    long long N, r = 0;
    cin >> N;
    while (N > 0) {
        r = r * 10 + N % 10;   // 取个位拼到 r 的末尾
        N /= 10;
    }
    cout << r << endl;          // 整数 r 自动去掉前导零
    return 0;
}
要点：用 r = r*10 + N%10 逐位反向拼接。由于 r 是整数，反转后的前导零会自动消失（如 1200 → 0021 → 21）。N 最大 20 亿，用 long long。
