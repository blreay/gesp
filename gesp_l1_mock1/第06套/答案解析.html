五、编程题参考程序
编程题 1：鸡兔同笼
#include <iostream>
using namespace std;
int main() {
    int h, f;
    cin >> h >> f;
    // 设兔 t 只，鸡 (h-t) 只：2*(h-t)+4*t = f  =>  t = (f-2*h)/2
    int t = (f - 2 * h) / 2;   // 兔
    int c = h - t;             // 鸡
    cout << c << " " << t << endl;
    return 0;
}
要点：由方程组直接解出兔数 t=(f-2h)/2，鸡数 c=h-t。也可用循环枚举鸡的只数 0~h，找到脚数匹配的解。
编程题 2：数列求和
#include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    double s = 0;
    for (int i = 1; i <= n; i++)
        s += 1.0 / i;      // 注意写 1.0 / i 而不是 1 / i
    printf("%.2lf\n", s);
    return 0;
}
易错点：必须写 1.0 / i，若写 1 / i 则是整数除法，除 i=1 外都得 0。累加用 double，%.2lf 保留两位。
