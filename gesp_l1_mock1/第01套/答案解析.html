三、单选题解析
第 1 题 C
变量命名规则：由字母、数字、下划线组成，不能以数字开头（A 错），不能含连字符 -（B 错），区分大小写故 Sum 与 sum 是两个不同变量（D 错）。C 正确。
第 2 题 B
double 是基本数据类型关键字。main 是函数名（标识符），printf、cout 是库提供的函数/对象，都不是关键字。
第 3 题 A
单引号括起的是字符常量（char），双引号括起的才是字符串。'A' 是 char 类型。
第 4 题 B
整除 17 / 5 = 3，取模 17 % 5 = 2，相加得 3 + 2 = 5。
第 5 题 A
* 和 % 优先级高于 +，从左到右先算 3 * 4 = 12，再 12 % 5 = 2，最后 2 + 2 = 4。
第 6 题 A
格式串中的 %d 依次被后面的参数 n（7）和 n*2（14）替换，其余原样输出，得 n=7, n*2=14。
第 7 题 B
非零为真、零为假。A：1&&0=0；B：1&&(!0)=1&&1=1（真）；C：!1=0；D：1-1=0。只有 B 为真。
第 8 题 B
累加 1+2+3+4+5 = 15。
第 9 题 B
i 取值 3、7、11、15、19，下一次 23≥20 停止，共 5 次。
第 10 题 A
偶数被 continue 跳过，只输出奇数：1#3#5#。
第 11 题 C
偶数即能被 2 整除，余数为 0，即 N % 2 == 0。D 判断的是奇数，A 同样判断奇数，B 错误。
第 12 题 A
关系运算结果用 1/0 表示。5>2 为 1，5==2 为 0，5<2 为 0，连续输出得 100。
第 13 题 B
1~100 中能被 10 整除的有 10,20,…,100，共 10 个。
第 14 题 C
“存储程序”原理由冯·诺依曼提出，称为“冯·诺依曼结构”。图灵奖以图灵命名；摩尔提出摩尔定律；香农创立信息论。
第 15 题 C
CPU 是“大脑”，由运算器+控制器+寄存器组成（C 对）。RAM 断电丢失（A 错）；鼠标是输入设备（B 错）；访问速度寄存器>高速缓存>内存>外存（D 错）。

五、编程题参考程序
编程题 1：班级平均分
#include <cstdio>
using namespace std;
int main() {
    int n;
    scanf("%d", &n);
    int sum = 0, x;
    for (int i = 0; i < n; i++) {
        scanf("%d", &x);
        sum += x;
    }
    printf("%.2lf\n", (double)sum / n);
    return 0;
}
要点：累加求和后转成 double 再除，避免整数除法丢失小数；用 %.2lf 保留两位小数。
编程题 2：数字位数和
#include <iostream>
using namespace std;
int main() {
    long long N;
    cin >> N;
    int sum = 0;
    while (N > 0) {
        sum += N % 10;   // 取出个位
        N /= 10;         // 去掉个位
    }
    cout << sum << endl;
    return 0;
}
要点：用 % 10 取个位、/ 10 去个位，循环到 N 为 0。注意 N 可达 10 亿，用 long long 更稳妥（int 也够，但养成好习惯）。
