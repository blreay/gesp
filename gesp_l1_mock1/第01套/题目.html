编程题 1：班级平均分

时间限制：1.0 s　|　内存限制：256.0 MB

题目描述

一个班级有 n 名同学，老师想知道这次考试全班的平均分（保留两位小数）。请你编写程序，读入 n 名同学的成绩，输出全班的平均分。

输入格式

第一行一个正整数 n，表示同学人数。
第二行 n 个用空格分隔的整数，表示每名同学的成绩。

输出格式

一行，一个小数，表示全班平均分，保留两位小数。

样例输入 1

3 80 90 100

样例输出 1

90.00

样例输入 2

4 60 75 88 91

样例输出 2

78.50

数据范围

对于所有测试点，保证 1 ≤ n ≤ 100，0 ≤ 每名同学成绩 ≤ 100。

编程题 2：数字位数和

时间限制：1.0 s　|　内存限制：256.0 MB

题目描述

给定一个正整数 N，请你计算它的各位数字之和。例如 N = 1234，则各位数字之和为 1 + 2 + 3 + 4 = 10。

输入格式

一行，一个正整数 N。

输出格式

一行，一个整数，表示 N 的各位数字之和。

样例输入 1

1234

样例输出 1

10

样例输入 2

9

样例输出 2

9

数据范围

对于所有测试点，保证 1 ≤ N ≤ 1,000,000,000。
