三、单选题解析
第 1 题 B
A 含关键字 int 不能作变量名；C 含以数字开头的 2sum；D 含非法符号 $。B 中三个名字全部由字母/数字/下划线组成且不以数字开头，合法（下划线开头也允许）。
第 2 题 A
* % / 同级高于 + -，从左到右：2*3=6，6%4=2，5/2=2。原式 = 10 - 2 + 2 = 10。
第 3 题 B
关键陷阱：a / b 都是 int，先做整数除法 7/2=3，再赋给 double 变成 3.0。%.1lf 输出 3.0。
第 4 题 C
ch + 1 中 char 参与算术运算时按 ASCII 码值，65 + 1 = 66，用 %d 按整数输出，得 66。
第 5 题 A
后置 x++ 先用原值参与运算：y = 5 + 2 = 7，之后 x 自增为 6。输出 6 7。
第 6 题 A
&& 优先级高于 ||。先算 a<b(1) && b>c(1) = 1，再 1 || (a>c)(0) = 1。逻辑结果用 1 表示。
第 7 题 B
内层循环对每个 i 执行 i 次，s = 1+2+3+4+5 = 15。
第 8 题 A
偶数被 continue 跳过；只累加奇数，到 i=13 时 13>13 不成立仍累加；下一个奇数 15 触发 i>13 break。所以 n = 1+3+5+7+9+11+13 = 49。
第 9 题 C
阶乘是连乘，应 f *= i。初值 f=1 才正确。
第 10 题 A
i 取 1,2,4,8,16,32,64（128>100 停止），累加 1+2+4+8+16+32+64 = 127。
第 11 题 B
闰年条件：（能被 4 整除 且 不能被 100 整除）或（能被 400 整除）。即 y%4==0 && y%100!=0 || y%400==0。&& 先算，正好对应规则。
第 12 题 A
a>5 为真，b = a-5 = 5；b%2==0？5%2=1 为假，c = 200。输出 5 200。
第 13 题 A
6 的因数有 1、2、3、6，共 4 个。
第 14 题 D

do...while 先执行后判断，依次输出 1 2 3 4 5。

五、编程题参考程序
编程题 1：分糖果
#include <iostream>
using namespace std;
int main() {
    long long m, n;
    cin >> m >> n;
    long long each = m / n;        // 每人分到
    long long left = m % n;        // 老师留下
    long long need = (left == 0) ? 0 : (n - left); // 凑齐还需
    cout << each << " " << left << " " << need << endl;
    return 0;
}
要点：/ 求每人份额、% 求剩余；"再凑多少恰好分完"= 当余数非 0 时为 n - 余数，余数为 0 时为 0（不要忘记这个边界）。
编程题 2：数位统计
#include <iostream>
using namespace std;
int main() {
    long long N;
    cin >> N;
    int total = 0, even = 0;
    while (N > 0) {
        int d = N % 10;            // 取出个位数字
        total++;
        if (d % 2 == 0) even++;    // 0/2/4/6/8 都是偶数数字
        N /= 10;
    }
    cout << total << " " << even << endl;
    return 0;
}
要点：边取个位边统计；注意 0 也算偶数数字。N 可达 20 亿，超过 int 上限（约 21 亿）边缘，用 long long 更安全。
