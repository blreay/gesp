🧪 编程题测试程序怎么用？

每套的两道编程题各配了一个 C++ 评测程序（在对应子目录里），用于自动校验你写的代码是否正确：

  1. 把你自己的代码编译成可执行文件：g++ my_answer.cpp -o my_answer
  2. 编译评测程序：g++ 编程题1_xxx_测试程序.cpp -o judge
  3. 运行（参数为你的可执行文件路径）：./judge ./my_answer（Windows：judge.exe my_answer.exe）
  4. 全部通过显示 ALL PASSED；不通过会用表格列出该用例的「输入 / 实际输出 / 期望输出」。
在线判卷：打开任意「题目」页，点选答案后点右上角红色【判卷】按钮，得分用大红字显示。
未做提醒：若有客观题未作答，判卷时会弹窗列出题号，只对已作答的题目计分。
错题复制：判错的题会高亮并显示正确答案，旁边【复制】按钮可把整题（题干+选项+你的答案+正确答案）复制到错题本。
