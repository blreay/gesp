五、编程题参考程序
编程题 1：奇数求和
#include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    long long s = 0;
    for (int i = 1; i <= n; i += 2)   // 直接每次 +2 遍历奇数
        s += i;
    cout << s << endl;
    return 0;
}
要点：从 1 开始每次 +2 即可只遍历奇数；也可遍历全部数用 if(i%2==1) 筛选。和可能较大，用 long long 更稳妥。
编程题 2：成绩等级
#include <iostream>
using namespace std;
int main() {
    int s;
    cin >> s;
    if (s >= 90) cout << "A";
    else if (s >= 80) cout << "B";
    else if (s >= 70) cout << "C";
    else if (s >= 60) cout << "D";
    else cout << "E";
    cout << endl;
    return 0;
}
要点：用 if-else if 链从高到低判断。注意顺序：先判 ≥90，再 ≥80……这样每个区间只会命中一个分支。
