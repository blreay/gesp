编程题 1：阶梯电价

时间限制：1.0 s　|　内存限制：256.0 MB

题目描述

某地实行阶梯电价：当月用电量不超过 200 度的部分，每度 0.5 元；超过 200 度的部分，每度 0.8 元。

给定小明家某月的用电量 x（度），请你计算这个月应缴的电费（保留两位小数）。

输入格式

一行，一个整数 x，表示用电量（度）。

输出格式

一行，一个小数，表示应缴电费，保留两位小数。

样例输入 1

100

样例输出 1

50.00

样例输入 2

300

样例输出 2

180.00

样例说明 2

前 200 度按 0.5 元 = 100 元，超出的 100 度按 0.8 元 = 80 元，合计 180.00 元。

数据范围

对于所有测试点，保证 0 ≤ x ≤ 100000。

编程题 2：第一个完全平方数

时间限制：1.0 s　|　内存限制：256.0 MB

题目描述

给定一个正整数 N，请你找出大于等于 N 的最小的完全平方数。一个数是完全平方数，当且仅当它能写成某个正整数的平方（如 1、4、9、16、25……）。

例如 N = 10，大于等于 10 的最小完全平方数是 16（4×4）。

输入格式

一行，一个正整数 N。

输出格式

一行，一个整数，表示大于等于 N 的最小完全平方数。

样例输入 1

10

样例输出 1

16

样例输入 2

25

样例输出 2

25

数据范围

对于所有测试点，保证 1 ≤ N ≤ 1,000,000。
