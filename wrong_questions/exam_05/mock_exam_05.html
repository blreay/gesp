15. 下面 C++ 代码先判断 6 是否为完全数，再用辗转相除法求 48 和 18 的最大公约数，最终输出是（ ）。

for (int i = 1; i < n; i++) {
    if (n % i == 0) sum += i;
}
int a = 48, b = 18;
while (b != 0) {
    int r = a % b;
    a = b;
    b = r;
}

    cout << "Y" << a;
else
    cout << "N" << a;

B. N6
C. Y12
D. Y48
